# MVP方案 — 最小可行产品

## 一句话

**打开编辑器，写纯文本，AI自动加格式和链接，保存后能搜到。**

4-6周交付。砍掉一切非必要功能。

---

## MVP只做三件事

```
1. 写 — 纯文本输入，零markdown知识
2. 自动加格式 — AI实时把纯文本变结构化markdown
3. 搜索 — 全文搜索已保存的笔记
```

**明确不做的事：**
- ❌ agent loop / ReAct
- ❌ 向量库 / embedding
- ❌ 知识图谱可视化
- ❌ MCP Server
- ❌ 健康检查 / Lint
- ❌ 语音输入
- ❌ 协作 / 同步
- ❌ 降重 / 论文生成（GradPaper那种）
- ❌ 拖拽Ingest
- ❌ 自动标签

### 核心流程：自动结构化

```typescript
// 伪代码 — 用户打字时的处理流程

async function onParagraphComplete(text: string) {
  // 1. 调LLM做结构化
  const response = await llm.chat({
    model: "deepseek-chat",
    messages: [
      {
        role: "system",
        content: `你是markdown格式化助手。
把用户的纯文本整理为结构化的markdown：
- 段落加合适的标题
- 关键信息加粗
- 列表自动编号
- 不要改变原意，不要添加内容
- 如果明显是笔记/会议记录/日记，加对应格式`
      },
      { role: "user", content: text }
    ],
    // 流式输出，实时渲染到编辑器
    stream: true
  });

  // 2. 流式更新TipTap编辑器
  for await (const chunk of response) {
    editor.updateParagraph(chunk);
  }
}
```

### 核心流程：搜索

```sql
-- SQLite FTS5 建索引
CREATE VIRTUAL TABLE notes_fts USING fts5(
  title, content, path,
  tokenize='unicode61'  -- 支持中文
);

-- 搜索
SELECT title, snippet(notes_fts, 1, '<mark>', '</mark>', '...', 20)
FROM notes_fts
WHERE notes_fts MATCH '王总 产品方向'
ORDER BY rank
LIMIT 10;
```

### 自动链接（可选，如果时间允许）

```typescript
// 保存笔记时，扫描已有笔记找重合关键词
async function autoLink(newNote: string, existingNotes: Note[]) {
  const response = await llm.chat({
    model: "deepseek-chat",
    messages: [
      {
        role: "system",
        content: `你是链接助手。给定一段新文本和已有笔记标题列表，
找出新文本中哪些词/概念跟已有笔记匹配，
返回JSON: {links: [{text: "AI编辑器", notePath: "会议记录-0712.md"}]}`
      },
      {
        role: "user",
        content: `新文本: ${newNote}\n\n已有笔记: ${existingNotes.map(n => n.title).join('\n')}`
      }
    ],
    response_format: { type: "json_object" }
  });

  return JSON.parse(response).links;
}
```

## 开发计划（4-6周）

### Week 1-2: 基础编辑器

**目标：能打开、写字、保存、看到列表**

```
✅ Tauri 2 + React项目搭建
✅ TipTap编辑器集成（WYSIWYG模式）
✅ 笔记CRUD（创建/读取/更新/删除）
✅ 本地文件系统读写（.md文件）
✅ 笔记列表侧边栏
✅ 自动保存
```

**交付物：** 一个能用的纯文本笔记编辑器（还没有AI功能）

### Week 3-4: AI结构化

**目标：写了纯文本，AI自动加格式**

```
✅ DeepSeek API接入（Vercel AI SDK）
✅ 设置页（API Key配置）
✅ 实时结构化引擎（段落级，debounce 2秒）
✅ 流式渲染到TipTap
✅ 结构化开关（用户可关闭AI）
✅ 自动标题生成
```

**交付物：** 写纯文本 → AI自动格式化，核心卖点可用

### Week 5-6: 搜索 + 自动链接

**目标：能搜到历史笔记，笔记间有链接**

```
✅ SQLite FTS5全文搜索
✅ 搜索结果页（高亮关键词）
✅ 自动链接（保存时扫描已有笔记，加[[]]链接）
✅ [[链接]]点击跳转
✅ 基础排版优化（字体/间距/暗色模式）
✅ 打包发布（macOS + Windows）
```

**交付物：** 可日常使用的MVP产品
